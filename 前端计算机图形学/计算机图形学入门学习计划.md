# 第一章 搭建开发环境


## 1.1 安装TypeScript

```
cnpm install typescript
```
![](img/1.png)

添加tsconfig.json

```json
{
    "compilerOptions": {
      "module": "commonjs",
      "target": "es6",
      "sourceMap": true
    },
    "exclude": [
      "node_modules"
    ]
  }
```

## Install webpack and ts dependencies

```
cnpm install webpack webpack-cli ts-loader --save-dev
```
